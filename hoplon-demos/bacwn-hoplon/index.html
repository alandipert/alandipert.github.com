Bacwn + Hoplon = ♡

This proof-of-concept demonstrates the use of Hoplon and bacwn together to create a ClojureScript web application. A small bacwn database of application state is queried using Datalog, reactively, to render the page content.

Source: https://gist.github.com/alandipert/8732704

Note: Once this prototype is cleaned up a bit, you'll be able to find it in the Hoplon Demos Repository.
